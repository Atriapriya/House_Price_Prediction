# 🏠 House Price Prediction — Streamlit App

## Folder Structure
```
house-price-streamlit/
├── app.py                  ← Streamlit UI
├── save_model.py           ← Run this in your notebook first
├── requirements.txt
├── model/
│   ├── model.pkl           ← Generated by save_model.py
│   └── scaler.pkl          ← Generated by save_model.py
└── house_price_regression_dataset.csv   ← Your dataset (optional for app)
```

---

## Step 1 — Save Your Model

Add this cell at the end of your Jupyter notebook and run it:

```python
import pickle, os
os.makedirs("model", exist_ok=True)

with open("model/model.pkl", "wb") as f:
    pickle.dump(lr, f)        # or `knn` if that's your best model

with open("model/scaler.pkl", "wb") as f:
    pickle.dump(scaler, f)
```

## Step 2 — Push to GitHub

```bash
git init
git add .
git commit -m "House price predictor app"
git remote add origin https://github.com/YOUR_USERNAME/house-price-predictor.git
git push -u origin main
```

---

## Step 3 — Deploy on Streamlit Cloud

1. Go to https://share.streamlit.io
2. Sign in with GitHub
3. Click **"New app"**
4. Select your repo → branch `main` → file `app.py`
5. Click **Deploy** ✅

Your app will be live at:
`https://YOUR_USERNAME-house-price-predictor.streamlit.app`

---

## Step 4 — Auto-redeploy

Every `git push` to `main` will automatically redeploy the app.
